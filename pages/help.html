Help

This should server as a basic guide to the most asked questions. For further details or other questions, contact the admins in the Discord.

How do I join the server?
Simply install the modpack, then boot up Minecraft and type in the IP. You can use other tutorials to find out how to do this.

What OSes do you support?
The modpack is built for Java edition. This means we support Windows, macOS, and Linux. Sorry mobile and console users.

I have an issue. Where can I report it?
Depends on what you have an issue with. Generally:

  • Issues with the Minecraft server go to the Elysia MC staff.
  • This is also true with Discord server issues.
  • Modpack issues go to ZeBoss.
  • Wiki and website issues go to daysant.

Are the modpack and the server related?
Not directly. While it is true they have a similar name and use case, the SMP and modpack are not directly tied to eachother.
